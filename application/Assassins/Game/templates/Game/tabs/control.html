Game Controls

  • Delete this game

    Deleting this game is irreversible and affects all participants!

  • Open / Close Registration Link

    By closing registration any distributed join links will cease to work.

  • Manage Players

    As a host you have the ability to remove players.

  • Manage Reviewers

    You can add additional reviewers to help manage the game.

  • Subscription

    If you want to allow more players to register you can update your subscription.
